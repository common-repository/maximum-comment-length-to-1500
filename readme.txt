== Description ==

Your users often leave long comments or copy long text to your comments and hide important comments.
If you are using standard wordpress comments system, this plugin can limit the maximum comments length to 1500 characters.

[Demo](https://rentrating.net/review/australia/queensland/kangaroo-point/19-castlebar-st-kangaroo-point-queensland-australia/).

== Installation ==

1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Maximum Comment Length to 1500' 
3. Click 'Install Now' and then activate the plugin
